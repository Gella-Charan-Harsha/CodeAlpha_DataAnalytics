# CodeAlpha Data Analytics Project

## Overview

This project was completed as part of the CodeAlpha Data Analytics Internship. The objective was to perform end-to-end data analysis by collecting data, exploring patterns, creating visualizations, and performing sentiment analysis using Python.

## Tasks Completed

### Task 1: Web Scraping

* Scraped 1000 book records from BooksToScrape using Python, Requests, and BeautifulSoup.
* Extracted book titles, prices, and ratings.
* Created a structured dataset for analysis.

### Task 2: Exploratory Data Analysis (EDA)

* Explored dataset structure, data types, and statistical summaries.
* Identified trends, patterns, and potential anomalies.
* Analyzed book prices and rating distributions.
* Performed correlation and outlier analysis.

### Task 3: Data Visualization

* Built professional visualizations using Matplotlib and Seaborn.
* Created rating distribution charts, price distribution plots, correlation heatmaps, violin plots, and outlier analysis charts.
* Developed visual insights to support data-driven decision-making.

### Task 4: Sentiment Analysis

* Generated and analyzed Amazon-style product reviews.
* Applied Natural Language Processing (NLP) using NLTK VADER Sentiment Analyzer.
* Classified reviews into Positive, Neutral, and Negative sentiments.
* Visualized sentiment distributions, category-wise sentiment trends, and review statistics.

## Dataset Information

### Books Dataset

* Total Records: 1000
* Features:

  * Title
  * Price
  * Rating

### Amazon Reviews Dataset

* Total Records: 5000+
* Features:

  * Review ID
  * Product Name
  * Category
  * Rating
  * Review Text
  * Review Date
  * Sentiment

## Key Findings

* The books dataset contained no missing values and was suitable for analysis.
* Average book price was approximately £35.
* Rating distributions revealed varying customer preferences.
* Sentiment analysis successfully classified customer opinions into positive, neutral, and negative categories.
* Visualizations provided meaningful insights into product ratings, pricing patterns, and customer sentiment trends.
